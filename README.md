# Language Translation Tool 🌐

## Overview
This is a real-time Language Translation Tool developed as part of the **CodeAlpha Artificial Intelligence Internship**. The tool allows users to input text in one language and translate it into a target language of their choice using the Google Translate API.

## Features
- **User-Friendly Interface**: Built with Streamlit for a smooth web experience.
- **Multiple Languages**: Supports all major languages provided by the Google Translate API.
- **Real-time Translation**: Quick and accurate translation at the click of a button.
- **Source Language Detection**: Automatically handles English and other source languages.

## Technologies Used
- **Python**: Core programming language.
- **Streamlit**: For the web-based User Interface (UI).
- **Googletrans (4.0.0-rc1)**: For accessing translation services.
